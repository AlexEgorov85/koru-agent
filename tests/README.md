# Система тестирования

## Структура тестов

В проекте реализована упрощенная, но надежная система тестирования:

```
tests/
└── verified/                 # Проверенные и работающие тесты
    ├── test_module_reload_functionality.py    # Тест перезагрузки модулей
    ├── test_planning_skill.py                 # Тест навыка планирования
    └── test_strategy_support.py               # Тест поддержки стратегий
```

## Текущее состояние

На данный момент в системе находятся только проверенные и работающие тесты:

1. **test_module_reload_functionality.py** - проверяет функциональность перезагрузки модулей
2. **test_planning_skill.py** - проверяет работу навыка планирования
3. **test_strategy_support.py** - проверяет поддержку стратегий в навыках

Все тесты проходят успешно без ошибок.

## Запуск тестов

Для запуска всех тестов:

```bash
python -m pytest tests/ -v
```

Для запуска конкретного теста:

```bash
python -m pytest tests/verified/test_strategy_support.py -v
```

## Примечание

Большинство тестов были удалены из-за ошибок и проблем с совместимостью. В дальнейшем планируется постепенное добавление новых, корректно работающих тестов.
