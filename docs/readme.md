# Composable AI Agent Framework

Этот проект — **инфраструктура для построения управляемых AI-агентов**, а не просто LLM-обёртка.

Ключевая идея:
> LLM **не управляет системой напрямую**.  
> Он принимает решения, которые **строго валидируются и исполняются кодом**.

## Для чего подходит
- автономные агенты
- tool-using агенты
- сложные reasoning-пайплайны
- агентные системы с контролем ошибок
- исследования (ReAct, Plan-Act-Reflect, Tree of Thoughts)

## Для чего НЕ подходит
- чат-боты “в лоб”
- простые prompt → response сценарии
