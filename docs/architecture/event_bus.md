# Event Bus

Event Bus — центральный механизм коммуникации.

## Типы событий
- TASK_EXECUTION
- PROGRESS
- USER_INTERACTION
- ERROR
- SYSTEM
- DEBUG

## Особенности
- Асинхронная доставка
- Глобальные и локальные подписчики
- Обработчики без знания источника события

Используется для:
- логирования
- визуализации шагов агента
- мониторинга
