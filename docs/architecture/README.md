## 🏗️ Архитектурные принципы

1. **Разделение слоёв**: Infrastructure → Application → Session
2. **Изоляция**: Каждый агент имеет собственный контекст
3. **Предзагрузка**: Ресурсы загружаются до начала выполнения
4. **Контрактная валидация**: Все данные валидируются по схемам
