# ComposableAgent

ComposableAgent — основной исполнитель логики.

## Возможности
- Выполнение atomic actions
- Выполнение composable patterns
- Адаптация к доменам
- Работа в нескольких стратегиях

## Варианты
- ComposableAgent — полный
- SimpleComposableAgent — упрощённый

## Ключевые методы
- execute_atomic_action
- execute_composable_pattern
- adapt_to_domain
