# 📊 Система оценки качества агента

## 🎯 Численные метрики

### Основные метрики

| Метрика | Формула | Что измеряет | Диапазон |
|---------|---------|--------------|----------|
| **Success Rate** | `successful / total × 100` | % успешных тестов | 0-100% |
| **Avg Steps** | `total_steps / successful` | Среднее шагов на тест | 1-∞ |
| **Efficiency Score** | `max(0, 100 - (avg_steps - 1) × 20)` | Эффективность (1 шаг = идеально) | 0-100 |
| **Overall Score** | `success_rate × 0.7 + efficiency × 0.3` | Общая оценка | 0-100 |

### Шкала оценок

| Score | Grade | Emoji | Интерпретация |
|-------|-------|-------|---------------|
| **90-100** | ОТЛИЧНО | 🏆 | Агент работает превосходно |
| **75-90** | ХОРОШО | ✅ | Агент работает хорошо |
| **50-75** | УДОВЛЕТВОРИТЕЛЬНО | ⚠️ | Агент работает, но есть проблемы |
| **0-50** | ТРЕБУЕТ УЛУЧШЕНИЙ | ❌ | Критические проблемы |

## 🚀 Использование

### 1. Запуск бенчмарка

```bash
# Запустить все тесты (34 теста)
py -m scripts.cli.run_real_agent_benchmark

# Запустить первые 5 тестов (быстрая проверка)
py -m scripts.cli.run_real_agent_benchmark --limit 5

# Запустить только SQL тесты (26 тестов)
py -m scripts.cli.run_real_agent_benchmark --level sql
```

## 📈 Интерпретация метрик

### Success Rate

| Значение | Интерпретация |
|----------|---------------|
| **90-100%** | Отлично, агент надёжен |
| **75-90%** | Хорошо, есть небольшие проблемы |
| **50-75%** | Удовлетворительно, нужны улучшения |
| **<50%** | Критично, агент часто ошибается |

### Avg Steps

| Значение | Интерпретация |
|----------|---------------|
| **1.0-1.5** | Отлично, агент эффективен |
| **1.5-2.5** | Хорошо, оптимальное количество |
| **2.5-4.0** | Удовлетворительно, много лишних шагов |
| **>4.0** | Критично, агент зацикливается |

### Overall Score

| Значение | Действия |
|----------|----------|
| **90-100** | Поддерживать качество |
| **75-90** | Оптимизировать проблемные места |
| **50-75** | Требуется рефакторинг |
| **<50** | Критический рефакторинг необходим |

## 🎯 Целевые показатели

Для production системы рекомендуется:

| Метрика | Target | Minimum |
|---------|--------|---------|
| **Success Rate** | ≥90% | ≥75% |
| **Avg Steps** | ≤1.5 | ≤2.5 |
| **Efficiency Score** | ≥80 | ≥60 |
| **Overall Score** | ≥85 | ≥70 |

---

## 🔧 Настройка весов

Веса метрик можно изменить в `run_real_agent_benchmark.py`:

```python
# Текущие веса
overall_score = (success_rate * 0.7 + efficiency_score * 0.3)

# Если важнее успешность
overall_score = (success_rate * 0.9 + efficiency_score * 0.1)

# Если важнее эффективность
overall_score = (success_rate * 0.5 + efficiency_score * 0.5)
```

## 📝 Чеклист для оценки

Перед релизом проверьте:

- [ ] Success Rate ≥ 75%
- [ ] Avg Steps ≤ 2.5
- [ ] Overall Score ≥ 70
- [ ] Нет тестов с >5 шагами
- [ ] Нет критических ошибок

Если все чекбоксы ✅ — агент готов к production!
